# Email Automation Scripts

Automated email sending and management scripts.

## Features

- 📧 Send bulk emails
- 📋 Email templates
- 📊 Email tracking
- 🔄 Auto-responders
- 📬 Newsletter automation

## Requirements

```bash
pip install -r requirements.txt
```

## Usage

1. Edit `config.py` with your SMTP settings
2. Run `python send.py`
